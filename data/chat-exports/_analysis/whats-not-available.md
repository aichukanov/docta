# Материал для статьи «Чего нет в аптеках Черногории (и чем заменить)»

Источник: TG-чаты, проиндексированы `scripts/chat-index/`. Ссылки — `[чат #id дата]`
(med = МЕДИЦИНА|ВРАЧИ|ЧЕРНОГОРИЯ, fam = Родители и Дети). Полные выборки:
`_analysis/unavailable-meds.jsonl`, `_analysis/declared-gaps.jsonl`.

Статья ложится на две цели: SEO (запросы «чем заменить X в Черногории», «нет X в ЧГ» —
высокий интент, аудитория уже в стране) и связку с фичей кросс-страновых названий (§
prd/drug-cross-country-reference). Тон — YMYL-осторожный: «нет прямого аналога, спросите
у фармацевта по действующему веществу», без советов по замене.

## Подтверждено сообществом: чего нет в ЧГ

### Сорбенты
- **Энтеросгель, Полисорб** — «нет в продаже» [med #20541 2023-07-25], просят привезти
  [med #34996]. (Активированный уголь есть.)

### Антисептики / обработка ран
- **Хлоргексидин, Мирамистин** «в привычном виде» — нет [med #3586, #3957, #21348].
- **Зелёнка, фукорцин, каламин** — нет (боль при ветрянке у детей) [med #31011 2024-02-23].
- **Диоксидин** — нет [med #618].

### «Противовирусные» / иммуностимуляторы (культурный разрыв РФ↔мир)
- Класса нет: **Виферон, Арбидол, Циклоферон, Кагоцел, Полиоксидоний, Цитовир-3, Эргоферон**
  [med #23439 2023-09-16, #23469, #24093]. Хороший образовательный блок: их «нет нигде,
  кроме РФ/СНГ». Для детей — [fam #12545 «Тут нет противовирусных для детей»].
- Есть таргетные противовирусные (Tamiflu от гриппа) [med #24093].

### Антигистаминные
- **Супрастин** (особ. в ампулах), **Тавегил** — нет прямого аналога по действующему
  веществу [med #3570, #19093]. Есть аналог супрастина — Senopen (мазь/инъекции).
- **Цетиризин / Зиртек** — «очень сложно найти» [med #3586]; подтверждено Anton: нет.
  Есть лоратадин/дезлоратадин/фексофенадин.
- **Антигистаминные капли** — нет, только таблетки/сиропы [med #3586].
- **Фенистил** — «говорят, тут нет» [med #58864].

### Регидратация
- **Регидрон** — аптечного нет (подтверждено Anton). Есть спортивные саше, но **с сахаром**
  — при кишечной инфекции это минус.

### «Домашние» сердечно-сосудистые / успокоительные
- **Корвалол, Валидол** — нет; **ввоз спорен** (Корвалол = фенобарбитал, в ряде стран
  контролируется) [med #21066 «ввозить вроде запрещено»]. YMYL/юр-нюанс — упомянуть аккуратно.

### Комбинированные анальгетики
- **Пенталгин** — нет аналога [med #20986]. (Соотносится с решением: комбинации — не в этой
  итерации фичи.)

### Прочее точечное
- **Сибутрамин** — нет ни под каким названием [med #24096].
- **Доктор МОМ, Каметон** — нет [med #3586].
- **Тобрекс** (тобрамицин, глазные) — вроде нет [med #19673].
- **ОК для кормящих (Чарозетта)** — нет [med #3586].
- **Моно-вакцины** — нет [fam #3746].

## Сквозная тема — не про «нет вещества», а про доступ (важный раздел статьи)
- **Антибиотики без рецепта не продают** — повторяется [med #11524, #33343, #101121].
  Русский бумажный/электронный рецепт не действует; спрашивать по **действующему веществу /
  по-латыни** [med #2782]. В Сербии/Боснии правила мягче — многие едут туда [med #2179, #24339].
- Многих препаратов нет и в соседних странах либо только по местному рецепту [med #21858].

## Не-медицинские «нет в ЧГ» (из семейного чата — для возможной broader-статьи)
Центральное отопление, моно-вакцины, гос-садики без очереди, русскоязычные средние школы,
детское такси с автокреслами — [fam #2575, #5129, #6949, #8888]. В медицинскую статью не
идут, но «Чего нет в Черногории» как рубрика может расширяться.

## Черновой план статьи
1. Вступление: почему привычной аптечки не хватает (формуляр ЧГ ≠ СНГ; рецептурные правила).
2. Чего реально нет + чем заменить (по действующему веществу, со ссылками на карточки
   /medicines и будущие страницы веществ).
3. Что нельзя/рискованно ввозить (Корвалол/фенобарбитал).
4. Как купить рецептурное (латынь/действующее вещество; соседние страны).
5. Дисклеймер: справка, не назначение.

Кандидаты в заголовок вещества-«якоря» для SEO: энтеросгель, регидрон, зиртек, мирамистин,
корвалол, супрастин, пенталгин, хлоргексидин.
