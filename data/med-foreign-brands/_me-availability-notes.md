# Заметки о наличии веществ в Черногории (для Фазы 2)

Ручные факты про вещества из `_deferred-phase2.json` (нет в CInMED-регистре).
Нужны, чтобы в Фазе 2 (страницы веществ / обратный поиск) корректно показывать
«нет в ЧГ» — только там, где это ПРАВДА, а не пробел данных.

Классификация «genuine» = реально не продаётся в ЧГ; «scrape-gap» = вероятно есть,
но отсутствует в нашем med_substances (НЕ утверждать «нет в ЧГ»).

| Вещество | Статус в ЧГ | Заметка |
|---|---|---|
| hydrogen peroxide (перекись водорода) | genuine | Только косметическая; как мед-средство в ЧГ не регистрируется. |
| oral rehydration salts (регидрон) | genuine | Аптечного регидранта нет. Есть только спортивные саше — **они с сахаром**, для регидратации при диарее/болезни это минус (сахар осмотически удерживает воду в кишечнике). Пометить при показе. |
| polymethylsiloxane polyhydrate (энтеросгель) | genuine | CIS-only. |
| benzyldimethyl-ammonium (мирамистин) | genuine | CIS-only. |
| validol | genuine | CIS-only. |
| brilliant green (зелёнка) | genuine | CIS-only. |
| phenazopyridine | genuine | US-only (Azo/Uristat). |
| drotaverine (но-шпа) | проверить | Восточноевропейский препарат; в ЧГ, вероятно, нет — подтвердить. |
| cetirizine (зиртек) | genuine | Подтверждено Anton: в ЧГ нет (несмотря на глобальную распространённость). Есть loratadine/desloratadine/fexofenadine. |
| chlorhexidine, guaifenesin, domperidone, melatonin, sennosides, clemastine, levocetirizine и др. | проверить | Уточнить по каждому перед показом «нет в ЧГ». |

Проверенные Anton отсутствия (перекись, регидрон, цетиризин) — все genuine. Похоже,
`med_substances` в порядке, а формуляр ЧГ просто отличается от СНГ. Отдельный аудит полноты
регистра пока НЕ нужен.
