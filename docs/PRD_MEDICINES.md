# PRD: Реестр лекарств Черногории

## Проблема

Экспаты и туристы в Черногории не знают:
- Есть ли их лекарство в черногорских аптеках
- Нужен ли рецепт
- Как называется аналог, если оригинал не зарегистрирован
- К какому врачу идти за рецептом

Единственный источник — сайт CInMED (cinmed.me), но он неудобен: без переводов, без поиска по действующему веществу, без аналогов.

Текущий раздел `/medications` (33 записи, привязаны к клиникам) слишком мал и не решает эту задачу.

## Решение

Заменить `/medications` на `/medicines` — полный реестр одобренных лекарств ЧГ с поиском, аналогами и переводами на 6 языков.

## Страницы

### 1. Список лекарств `/medicines`

**Поиск**: строка поиска по названию лекарства или действующего вещества. Ищет по всем языкам (`name`, `med_substances.name_en`, `med_substances.name_ru` и т.д.).

**Фильтры**:
- Режим отпуска: без рецепта / по рецепту / все (по умолчанию — все активные)
- ATC-группа: выпадающий список из 14 категорий ("Сердечно-сосудистая система", "Нервная система", ...)
- Только с действующей лицензией: чекбокс (по умолчанию — да)

**Карточка в списке**:
```
ASPIRIN® PROTECT
Gastro-resistant tablet, 100 mg
Bayer Bitterfeld GmbH, Германия
🟢 Без рецепта
```

**SEO**: title = "Лекарства в Черногории (2523)", description = "Полный реестр одобренных лекарств..."

### 2. Страница лекарства `/medicines/[slug]`

**Основная информация**:
- Название (бренд)
- Действующее вещество (INN) — с переводом на язык пользователя
- Форма выпуска, дозировка, упаковка
- Производитель, страна
- Режим отпуска — выделено, с иконкой
- ATC-группа

**Блок "Аналоги в Черногории"** (ключевая фича):
Все лекарства с тем же действующим веществом (INN), с активной лицензией.
```
Другие лекарства с парацетамолом в Черногории:
- EFFERALGAN® — раствор, 30mg/ml — без рецепта
- FEBRICET — таблетка, 500mg — без рецепта
- DORETA — таблетка, 37.5mg+325mg (+ трамадол) — по рецепту
```

**Блок "Нужен рецепт?"**:
- Без рецепта → "Продаётся свободно в любой аптеке Черногории"
- По рецепту → "Нужен рецепт врача" + ссылка на `/doctors` с фильтром по подходящей специальности
- Стационарное → "Только в медучреждении"

**Блок "Данные из реестра"**:
- Держатель лицензии
- Номер и дата разрешения
- Полный адрес производителя
- Ссылка на cinmed.me

**SEO**: title = "ASPIRIN PROTECT в Черногории — без рецепта, аналоги", description = "Аспирин Протект (ацетилсалициловая кислота, 100mg) одобрен в Черногории. Без рецепта. 15 аналогов..."

### 3. Страница вещества `/medicines/substance/[slug]`

Все лекарства с данным действующим веществом.

**Пример**: `/medicines/substance/paracetamol`
- Заголовок: "Парацетамол — 49 лекарств в Черногории"
- Описание вещества (INN на всех языках)
- ATC-группа
- Таблица: все лекарства, сгруппированные по форме выпуска

**Ценность**: отвечает на запрос "есть ли парацетамол в Черногории" и показывает все варианты.

### 4. Виджет "Проверка лекарства" на главной

Компактная строка поиска: "Проверьте, есть ли ваше лекарство в Черногории".
При вводе — автокомплит с быстрыми результатами, клик → страница лекарства.

## Поиск аналогов для RU/UA лекарств

Пользователь вводит "Цитрамон" → не найдено в реестре → показываем:
```
❌ "Цитрамон" не зарегистрирован в Черногории

Действующие вещества Цитрамона:
парацетамол + кофеин + ацетилсалициловая кислота

Доступные аналоги:
- CAFFETIN® (парацетамол + кофеин + пропифеназон + кодеин) — по рецепту
- FEBRICET COLD (парацетамол + кофеин + аскорбиновая кислота) — без рецепта
```

**Реализация**: отдельная JSON-таблица `data/ru-ua-medicines.json` с маппингом популярных RU/UA названий → INN. Не обязательно в первой версии — можно добавить позже.

## API

### `POST /api/medicines/list`

```ts
body: {
  name?: string        // поиск по названию/INN (LIKE по всем языкам)
  dispensingMode?: 'otc' | 'prescription' | 'all'
  atcGroupId?: number  // фильтр по ATC-группе
  activeOnly?: boolean // по умолчанию true
  locale: string
  page: number
}
response: {
  items: Medicine[]
  totalCount: number
}
```

### `POST /api/medicines/details`

```ts
body: { slug: string, locale: string }
response: {
  ...medicine fields,
  substances: SubstanceWithTranslations[],
  analogs: Medicine[],    // same INN, active, limit 20
  pharmaForm: { name, name_en, name_ru, ... },
  dispensingMode: { name, name_en, name_ru, ... },
  manufacturer: { name, full_address, country: { name_en, ... } },
}
```

### `POST /api/medicines/substance`

```ts
body: { slug: string, locale: string, page: number }
response: {
  substance: SubstanceWithTranslations,
  medicines: Medicine[],
  totalCount: number,
}
```

## SEO-стратегия

**Страницы лекарств** (2523 активных):
- Каждая — уникальная страница с уникальным title/description
- Ловит запросы: "aspirin montenegro", "buy ibuprofen budva", "амоксициллин в черногории"
- Schema.org: `Drug` с `activeIngredient`, `prescriptionStatus`

**Страницы веществ** (~500 с 2+ лекарствами):
- Ловит запросы: "paracetamol analog montenegro", "парацетамол аналоги черногория"

**Sitemap**: все активные лекарства + вещества с 2+ лекарствами.
