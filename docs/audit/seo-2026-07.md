# SEO-аудит docta.me — июль 2026

## Ситуация на момент аудита

Сайт упёрся в ~1000 показов/день при ~54k страниц в индексе (~0,02 показа/страницу/день).

**Цифры GSC:** ~54k страниц не в индексе (33 672 crawled-not-indexed, 11 418 noindex, 7 622 redirect, 830 404, 77 5xx), ~54k в индексе.

**Ссылочный профиль:** 1 081 внешняя ссылка, из них 972 — собственный busticket4.me (сквозная), остальные ~100 унаследованы через 301 с купленных брошенных доменов клиник (bolnica-meljine, domzdravljahercegnovi, ordinacijanatal, malbaski — анкоры это выдают). Заработанных ссылок ≈ 0.

**Внутренняя перелинковка:** топ по входящим — /privacy (40k), footer-страницы. На фильтрованные city×specialty страницы HTML-ссылок нет вообще — специальности упомянуты в doctor/specialties.vue как текст, крошки врача не поднимаются до уровня город/специальность.

## Что уже реализовано в коде (проверено на 2026-07)

- hreflang + canonical в app.vue (`getCanonicalUrl` — единая точка)
- sitemap с alternates и порогами (`server/common/sitemap/`)
- 301 с числовых ID и старых слагов (`server/middleware/redirect-routes.ts`)
- noindex на пустых списках (`list-page.vue`)
- Пагинация листингов написана, но по состоянию на 2026-07-04 не была закоммичена — уточнить текущий статус перед следующим шагом.

svad.net того же владельца с той же архитектурой даёт 10k показов/день → дело не в технике, а в YMYL-пороге (медицина) + нулевом доверии домена + thin-страницах.

## План (по приоритету)

1. **Перелинковка** — крошки с уровнем «специальность × город», специальности-ссылки, link-хабы на листингах.
2. **Задеплоить пагинацию** листингов.
3. **Сжать индексируемую поверхность** — политика фасетов, чистка 404/5xx, разбор примеров crawled-not-indexed.
4. **Контент-слой из БД** — справки на страницах услуг/анализов (что это / как проводится / когда назначают / подготовка / о чём может говорить отклонение), цены min/avg, FAQ. Старт — топ-50+ услуг и анализов по покрытию клиниками. См. [[service-reference-content]] (PRD `prd/service-reference-content/`).
5. **E-E-A-T** — about-страница с людьми, источники данных, приглушить сквозную связку с busticket4.me.
6. **Заработанные ссылки** — бейджи клиникам, экспатские сообщества (цель 30–50 доменов за полгода).
