## Краткое описание

Комплексная система аналитики с использованием **Mixpanel** (product analytics) и **Google Analytics 4** (marketing & SEO) для отслеживания поведения пользователей, конверсий и ключевых метрик платформы.

**Текущий статус:** 🔴 Not Started (0/4 итераций) - [Смотреть прогресс →](PROGRESS.md)

### Двойная система аналитики

🔵 **Mixpanel** - для product команды:

- Event-based analytics
- User funnels & cohorts
- Retention analysis
- User profiles

🟢 **Google Analytics 4** - для маркетинга:

- Traffic sources & SEO
- Google Ads integration
- Demographics
- Standard reports

### Ключевые возможности

📊 Отслеживание просмотров страниц (клиники, врачи, услуги, лекарства, статьи)  
🖱️ Трэкинг взаимодействий (клики, копирование контактов, переходы)  
🎯 Воронки конверсий (просмотр → контакт → переход на сайт)  
👥 Профили пользователей и сегментация  
📈 Когортный анализ и retention  
🔍 Источники трафика и SEO метрики  
🎨 A/B тестирование (подготовка инфраструктуры)

### Основные события для трэкинга

#### Просмотры страниц

- 📄 Просмотр страницы клиники
- 👨‍⚕️ Просмотр страницы врача
- 💊 Просмотр страницы услуги
- 💉 Просмотр страницы лекарства
- 📰 Просмотр статьи

#### Взаимодействия с контентом

- 📋 Копирование контактов (телефон, email, адрес)
- 🌐 Переход на сайт клиники
- 📱 Клик на телефон (call action)
- 📧 Клик на email
- 🔗 Переход с услуги на клинику
- 🔗 Переход с врача на клинику
- ⭐ Добавление в избранное

#### Навигация и поиск

- 🔍 Поисковые запросы
- 🏷️ Клик на тег/категорию
- 📍 Фильтр по локации
- 🗺️ Взаимодействие с картой

#### Конверсионные события

- 📝 Начало заполнения формы записи
- ✅ Завершение заполнения формы
- ⭐ Оставление отзыва
- 📱 Установка mobile app (будущее)

### Зависимости

- **Не критично** - может быть интегрировано на любом этапе
- **Рекомендуется** после базовых страниц клиник/врачей/услуг

### Что НЕ включено

❌ Heatmaps (Hotjar) - отдельный PRD  
❌ Session recording - отдельный PRD  
❌ Server-side analytics - пока только client-side  
❌ Custom analytics dashboard - используем Mixpanel UI + GA4 Dashboard  
❌ Google Ads интеграция - настроим после запуска рекламы

## Технологии

- **Analytics:** Mixpanel (client SDK) + Google Analytics 4
- **Frontend:** Nuxt 3 Plugin + Composables
- **Type Safety:** TypeScript события и свойства
- **Privacy:** Cookie consent интеграция

## Scope изменений

### Конфигурация

- `.env` - добавить `MIXPANEL_TOKEN`
- `nuxt.config.ts` - настройки приватности и runtime config

- `plugins/mixpanel.client.ts` - инициализация Mixpanel
- `composables/useAnalytics.ts` - хелперы для трэкинга
