## Краткое описание

Система самостоятельной покупки платных услуг для администраторов клиник с интеграцией платежной системы, аналогичной процессу покупки через админку.

**Текущий статус:** 🔴 Not Started (0/4 итераций) - [Смотреть прогресс →](PROGRESS.md)

### 🎯 Ключевые возможности

✅ **Просмотр услуг** - администратор клиники видит все доступные платные услуги  
✅ **Выбор услуг** - выбор нескольких услуг для покупки в одном заказе  
✅ **Период действия** - выбор периода действия услуг (1, 3, 6, 12 месяцев)  
✅ **Расчет стоимости** - автоматический расчет итоговой стоимости  
✅ **Оплата онлайн** - интеграция с платежной системой  
✅ **История покупок** - просмотр активных и истекших покупок  
✅ **Активация** - автоматическая активация услуг после оплаты

### Доступные услуги (из `billing_paid_services`)

1. **DOFOLLOW** (id: 1) - Ссылки dofollow (для SEO)
2. **HIGHLIGHT** (id: 2) - Выделение клиники в списках
3. **APPROVED** (id: 3) - Верификация клиники с бейджем ✓

### Что НЕ включено

❌ Оплата через админку (уже реализовано)  
❌ Подписки с автопродлением (будет в следующей версии)  
❌ Промокоды и скидки (будет в следующей версии)  
❌ Возврат платежей (будет в следующей версии)

## Scope изменений

### База данных

- Существующие таблицы (используются без изменений):
  - `billing_paid_services` - список доступных услуг
  - `billing_clinic_service_purchases` - покупки клиник
  - `billing_clinic_service_purchase_items` - элементы покупок
- Новые таблицы:
  - `billing_payment_transactions` - транзакции платежной системы

### API Endpoints

- `GET /api/billing/services/catalog` - каталог услуг для клиники (публичный)
- `POST /api/billing/orders/create` - создать заказ
- `GET /api/billing/orders/[id]` - детали заказа
- `POST /api/billing/orders/[id]/payment` - инициировать оплату
- `POST /api/billing/webhooks/payment` - webhook для подтверждения оплаты
- `GET /api/billing/purchases/my` - мои покупки (для администратора клиники)

### UI Компоненты

- `pages/profile/clinics/[id]/billing.vue` - страница платных услуг
- `components/billing/ServiceCard.vue` - карточка услуги
- `components/billing/OrderForm.vue` - форма заказа
- `components/billing/PaymentProcessor.vue` - обработка платежа
- `components/billing/PurchaseHistory.vue` - история покупок
