## Задачи

### 1. Создание главного компонента ScheduleEditor.vue

**Файл:** `components/clinic/working-hours/ScheduleEditor.vue`

**Функционал:**

- Загрузка текущего графика через API
- Редактирование графика для всех дней недели
- Кнопки быстрого заполнения:
  - "Копировать на все дни"
  - "Копировать на рабочие дни (Пн-Пт)"
  - "Копировать на выходные (Сб-Вс)"
  - "Установить 24/7"
- Сохранение через API
- Отмена изменений

### 2. Создание компонента DaySchedule.vue

**Файл:** `components/clinic/working-hours/DaySchedule.vue`

**Функционал:**

- Отображение названия дня
- Выбор типа дня (DayTypeSelector)
- Список временных интервалов (TimeRangeSelector)
- Кнопка "Добавить интервал" (до 3-х)
- Кнопка "Скопировать на другие дни"

### 3. Создание компонента TimeRangeSelector.vue

**Файл:** `components/clinic/working-hours/TimeRangeSelector.vue`

**Функционал:**

- Выбор времени начала (Element Plus TimePicker)
- Выбор времени окончания (Element Plus TimePicker)
- Кнопка удаления интервала
- Валидация: start < end

### 4. Создание компонента DayTypeSelector.vue

**Файл:** `components/clinic/working-hours/DayTypeSelector.vue`

**Функционал:**

- Radio buttons или Select для выбора типа:
  - Рабочий день (regular)
  - Круглосуточно (24/7)
  - Выходной (closed)
  - По требованию (on_demand)
  - Не указано (not_specified)

### 5. Интеграция в страницу редактирования клиники

**Файл:** `pages/profile/clinics/[id]/edit.vue`

Добавить вкладку "График работы" с компонентом `ScheduleEditor`.

### 6. Валидация на клиенте

Создать композабл для валидации:

**Файл:** `composables/useWorkingHoursValidation.ts`

## Как проверить

### Ручное тестирование

1. Открыть страницу редактирования клиники
2. Перейти на вкладку "График работы"
3. Выбрать тип "Рабочий день" для понедельника
4. Добавить интервал 08:00 - 20:00
5. Нажать "Копировать на рабочие дни"
6. Проверить, что Вт-Пт имеют такой же график
7. Для субботы добавить интервал 09:00 - 15:00
8. Для воскресенья выбрать "Выходной"
9. Нажать "Сохранить"
10. Проверить, что данные сохранились (перезагрузить страницу)

### Тестирование валидации

1. Попробовать добавить 4-й интервал (должна быть ошибка)
2. Установить start = 20:00, end = 08:00 (должна быть ошибка)
3. Добавить пересекающиеся интервалы (должна быть ошибка)
