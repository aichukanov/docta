## Задачи

### 1. Создание компонента ScheduleDisplay.vue

**Файл:** `components/clinic/working-hours/ScheduleDisplay.vue`

**Функционал:**

- Загрузка графика через API
- Отображение дней недели с графиком
- Выделение текущего дня
- Локализация названий дней
- Красивое форматирование интервалов

### 2. Создание компонента CurrentStatus.vue

**Файл:** `components/clinic/working-hours/CurrentStatus.vue`

**Функционал:**

- Расчет текущего статуса (открыто/закрыто)
- Отображение времени до закрытия/открытия
- Автоматическое обновление каждую минуту
- Иконка статуса (зеленая/красная)

### 3. Интеграция в публичную страницу клиники

**Файл:** `pages/clinics/[clinicId]/index.vue`

Добавить блок "График работы" с компонентами `CurrentStatus` и `ScheduleDisplay`.

### 4. Стилизация

**Файл:** `components/clinic/working-hours/ScheduleDisplay.module.scss`

Современный, свежий, легкий дизайн (следуя guidelines проекта).

### 5. Локализация

**Файлы:**

- `locales/ru.json`
- `locales/en.json`
- `locales/sr.json`
- и т.д.

Добавить переводы для:

- Названий дней недели
- Статусов ("Открыто", "Закрыто", "Круглосуточно", и т.д.)
- Сообщений ("Откроется в", "Открыто до", и т.д.)

## Как проверить

### 1. Проверка отображения

1. Открыть публичную страницу клиники (например, `/clinics/1`)
2. Найти блок "График работы"
3. Проверить, что отображаются все дни недели
4. Проверить, что текущий день выделен

### 2. Проверка статуса "Открыто/Закрыто"

**Тест 1: Клиника открыта (в рабочее время)**

1. Открыть страницу клиники с графиком Пн-Пт 08:00-20:00
2. Открыть в рабочее время (например, в 10:00 в понедельник)
3. Проверить, что статус "Открыто сейчас"
4. Проверить, что показывается "Открыто до 20:00"

**Тест 2: Клиника закрыта (в нерабочее время)**

1. Открыть страницу в нерабочее время (например, в 22:00)
2. Проверить, что статус "Закрыто"
3. Проверить, что показывается время открытия

**Тест 3: Клиника 24/7**

1. Открыть страницу клиники с типом "24/7"
2. Проверить, что статус "Открыто круглосуточно"

### 3. Проверка локализации

1. Переключить язык на русский → проверить переводы
2. Переключить язык на английский → проверить переводы
3. Переключить язык на сербский → проверить переводы

### 4. Проверка адаптивности

1. Открыть на десктопе → проверить дизайн
2. Открыть на планшете → проверить дизайн
3. Открыть на мобильном → проверить дизайн
