# Ростер «аптечка туриста» — черновик на утверждение

Статус: черновик · Дата: 2026-07-23 · Собрал: Claude · Утверждает: Anton

Назначение: курируемый список действующих веществ (INN), популярных у туристов, —
основа для `med_foreign_brands` (см. PRD.md §5). Только **однокомпонентные** вещества
(комбинированные — вне scope, см. хвост файла).

Колонки: INN — международное непатентованное наименование; «Бренд-маркер» — узнаваемое
торговое название (преим. русскоязычный рынок), по которому опознаётся позиция; «Флаги» —
YMYL/регуляторные пометки для этапа сбора. Наличие в ЧГ и полная матрица брендов
(RU/DE/PL/US) добираются в Фазе 1 — здесь НЕ проставлено.

Легенда флагов: `Rx` — за рубежом часто рецептурный; `!US` — нет/не одобрен в США;
`!EU` — ограничен/снят в ЕС; `?INN` — уточнить точное INN при сборе.

## Итог
Уникальных однокомпонентных позиций (после свёртки дублей #86/#88/#97): **~97**.
Довести до ровно 100 — резерв кандидатов ниже.

Резерв (докинуть при утверждении): пиридоксин (B6) отдельно, фолиевая к-та,
лоперамид+симетикон (combo — нет), эргоферон (не INN), лидокаин-спрей, декспантенол-глазной,
цетиризин-капли, ксилометазолин детский, натрия хлорид (физраствор), калия перманганат
(марганцовка, !US/!EU).

## Вне scope — комбинированные (для справки, НЕ в `med_foreign_brands`)
Показывать негде (нет карточки-носителя на одно вещество), но узнаваемы у туристов:
- Терафлю, Колдрекс, Фервекс — парацетамол + фенилэфрин + фенирамин ± вит.C
- Пенталгин — парацетамол/напроксен + кофеин + дротаверин + фенирамин
- Корвалол, Валокордин — фенобарбитал + этилбромизовалерианат (`!US`/`!EU` — фенобарбитал)
- Отипакс — лидокаин + феназон (ушные капли)
- Стрепсилс — амилметакрезол + дихлорбензиловый спирт
- Но-шпалгин, Спазмалгон — метамизол + спазмолитик
- Цитрамон — АСК + парацетамол + кофеин

## Каверзные позиции (обязательная проверка при сборе — YMYL)
Классические «нет прямого аналога за рубежом» — хорошие примеры для подсказки
«в Черногории/этой стране такого нет»:
- Мирамистин, Энтеросгель, Валидол, Зелёнка, Корвалол — по сути только СНГ-рынок.
- Но-шпа (дротаверин), Супрастин (хлоропирамин), метамизол (анальгин) — нет в США.
- Феназопиридин — есть в США (Uristat), нет в ЕС — обратный случай.
